# bricklink-tools

`sudo pip install -r requirements.txt`

Create a file called `secret` in `bricklink-tools` directory:

```
ConsumerKey,[YOUR VALUE]
ConsumerSecret,[YOUR VALUE]
TokenValue,[YOUR VALUE]
TokenSecret,[YOUR VALUE]
```
